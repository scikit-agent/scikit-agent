# Ecosystem

`scikit-agent` is an open source project that takes part in a larger ecosystem.

We see `scikit-agent` as a new entrant into the
[Scientific Python ecosystem](https://scientific-python.org/), and look to that
community for guidance and best practices.

Our scientific domain is the converging fields of computational economics,
agent-based modeling, and multi-agent systems.

We believe that high quality, open source tooling, using the best available
techniques from computer science, can advance these social scientific fields.
